## Sobre 

Este projeto foi idealizado como uma cópia das funções básicas de uma rede social. Nela o usuário consegue compartilhar textos e dar likes em determinados posts.

### tecnologias

- Vue.js 2
- Vue CLI
- Router
- Firebase 8
